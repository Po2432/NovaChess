Installation Guide (Play Anywhere Offline)

🍎 iOS (Safari)

Tap the Share button at the bottom, then select "Add to Home Screen".

🤖 Android (Chrome)

Tap the Menu (⋮) icon, then select "Install App" or "Add to Home Screen".

💻 Desktop (Chrome/Edge)

Click the Install icon (⊕) located on the right side of the browser URL bar.
